# 📝 NVC分解テンプレート
# テーマ：

妻「どうしてあなたはゲームばっかりしているの？」

# ① 観察（Observation）

※評価・解釈を入れない。録音したように。

妻が言った言葉：

**「どうしてあなたはゲームばっかりしているの？」**

その時の状況：

いつ： 日曜日の午前
どこで：　家で
自分は何をしていたか：　PCゲーム「DIVISION」をしていた

# ② 自分の感情（Feeling）

※思考ではなく感情語で。

第一反応の感情：

**「ちょっと待って、このボスが強すぎて手を離せない」**

少し時間が経ってからの感情：　

「夢中になってやってたから妻が何をしていたか見ていないな」
「朝の忙しい時間についやりたくてゲームをしてしまった」

（必要なら複数）

# ③ 自分のニーズ（Need）

※満たしたい大切な価値・欲求。

この場面で大事にしているもの：
- 自分の役割
- 妻の負担の軽減
- ゲームがしたい

# ④ 妻のニーズ（仮説）

※決めつけない。「かもしれない」で書く。

妻はもしかすると：
**僕に何かをしてほしいのに頼めなかったのではないか**

# ⑤ リクエスト（Request）

※命令ではなく、具体的で今ここで可能なもの。

A. 自分から伝えるなら

「　そういえば、おひなさまを設置するんだっけ？よしやろう。　」

B. 妻に確認するなら

「 ごめん、僕は何かすることを忘れていた？　」

# ⑥ メタ確認（任意）

※自分の誠実さチェック

これは説明になっていないか？： ◯

相手のニーズを否定していないか？：◯
   
自分のニーズを隠していないか？：△
