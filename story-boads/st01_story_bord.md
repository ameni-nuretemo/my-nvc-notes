# 📝 NVC分解テンプレート
# テーマ：

娘「なんでいじめがなくならないの？」

# ① 観察（Observation）

※評価・解釈を入れない。録音したように。

娘が言った言葉：

**「なんでいじめがなくならないの？」**

その時の状況：

いつ： 月曜日の朝
   
どこで：　家で
   
自分は何をしていたか：　家族でTVを見ながら朝食をとっていた

# ② 自分の感情（Feeling）

※思考ではなく感情語で。

第一反応の感情：焦り、戸惑い、緊張、

少し時間が経ってからの感情：困惑

# ③ 自分のニーズ（Need）

※満たしたい大切な価値・欲求。

この場面で大事にしているもの：期待に応えたい

# ④ 娘のニーズ（仮説）

※決めつけない。「かもしれない」で書く。

娘はもしかすると：不条理にいらだっているのかもしれない

# ⑤ リクエスト（Request）

※命令ではなく、具体的で今ここで可能なもの。

A. 自分から伝えるなら

世の中にはどうしても悪い感情や攻撃的な感情を持った人もいるんだよ。

B. 娘に確認するなら

周りにそのようなことが起こっている？

# ⑥ メタ確認（任意）

※自分の誠実さチェック

これは説明になっていないか？： ×

相手のニーズを否定していないか？：　△
   
自分のニーズを隠していないか？：　〇

この問題に答えはない。だけど無理に理解させるのではなく、娘と一緒に納得いくまで話し合うことがNVCだと思われる。
